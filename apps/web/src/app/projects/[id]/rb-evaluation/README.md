# RB Evaluation (RB 평가 검토)

## 개요

사업 공고의 **평가 항목**에 맞춰 자체 점검하는 페이지입니다. RB(Research-Based) 평가는 산·학·연 전문가로 구성된 평가단이 수행하며, 연구자가 **제출 전에 스스로 평가 기준에 맞춰 점검**하면 누락·약점을 보완할 수 있습니다.

## 왜 이렇게 만들었는지

### 평가 항목 기반 체크리스트

- research-analysis의 지원사업 요약에 있는 "핵심 평가 항목"을 그대로 활용
- 예: 기술적 창의성 30점, 연구수행능력 25점, 기대효과 25점, 추진계획 적정성 20점
- **자가점수 입력** → "내가 이 항목에서 몇 점 받을 것 같은가" 자기평가
- **체크리스트**: 각 항목별 세부 질문 (선행기술 대비 차별성, 정량적 목표 등)

### 미달 항목 하이라이트

- 배점의 70% 미달 시 amber 경고
- "보완 권장: 기술적 창의성, 추진계획 적정성" 등으로 항목명 표시
→ 약한 부분을 의식적으로 보완하도록 유도

### 세로 스크롤 단일 페이지

- 탭 없이 항목 순서대로 스크롤. 평가위원이 보는 순서와 동일하게 구성
- 상단 요약 → 본문 항목별 상세 → 하단 총점·미달 요약
