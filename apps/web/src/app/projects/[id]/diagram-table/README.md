# Diagram/Table (도식/표 제안)

## 개요

연구계획서 본문을 바탕으로 **필요한 도식·표**를 추천하고 AI가 초안을 생성하는 페이지입니다. 연구자가 과거 생성 내역을 관리하면서 체계적으로 도식화·개선할 수 있도록 구성합니다.

## 레이아웃

### 3단 리사이징 (react-resizable)

| 구역 | 설명 |
|------|------|
| **좌측** | 도식/표 목록 (1칼럼). 카드: [썸네일 \| 캡션명 + 생성일]. 과거 내역 선택 |
| **중앙** | 채팅. AI와 대화하며 도식 생성·수정·개선 요청 |
| **우측** | 미리보기. 선택된 도식/표의 현재 버전 표시 |

좌측↔중앙, 중앙↔우측 경계를 드래그하여 패널 너비 조절 가능.
