# Tech Standard (기술표준분류)

## 개요

**국가과학기술표준분류(NESTI)**에 맞춰 기술코드를 검색·선택하는 페이지입니다. 연구계획서에는 우선순위별 소분류 코드명과 비중(%)을 1~3순위로 기재해야 합니다.

## 왜 이렇게 만들었는지

### 검색 + 좌우 분할

- **상단 검색창**: 키워드로 소분류 검색. NESTI 구조가 깊어서 직접 찾기 어렵기에 검색 필수
- **추천 검색어**: 연구주제·키워드를 기반으로 초기 검색어 제안 → "컴퓨터비전", "건설안전" 등
- **좌측**: 검색 결과 리스트 (코드, 분류명, 경로)
- **우측**: 선택한 분류 (1~3순위, 비중 %)

### 1~3순위, 비중 합 100%

- 공고에서 "우선순위별 소분류 코드명과 비중(%)"을 요구
- 1순위가 가장 비중 높음. 기본값 50/30/20으로 제안
- 합계 100% 검증 → amber 경고로 합계가 100이 아님을 표시

### 선택 UX

- 검색 결과에서 클릭 시 **첫 번째 빈 순위**에 자동 할당
- 같은 분류 중복 선택 방지
- 비중은 숫자 입력으로 조정

## NESTI 분류 구조 (참고)

- 대분류 > 중분류 > 소분류 계층
- 코드는 6자리 (예: 350101)
- 실제 연동 시 NTIS 등 공개 API 또는 DB 사용
