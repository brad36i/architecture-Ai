# Research Plan (연구계획서 작성)

## 개요

연구계획서 **초안** 작성 페이지. 채팅 기반으로 AI와 대화하며 초안을 생성하고, 캔버스에서 영역을 지정해 수정 요청을 할 수 있다.

## 레이아웃

```
┌─────────────────┬─────────────────────────────┐
│                 │                             │
│   채팅 패널     │   연구계획서 초안 캔버스    │
│   (왼쪽)       │   (오른쪽)                   │
│                 │                             │
│   - 메시지 목록 │   - 초안 텍스트 표시        │
│   - 입력창      │   - 영역 선택 → 액션        │
│                 │     · 다시 물어보기         │
│                 │     · 새로쓰게              │
└─────────────────┴─────────────────────────────┘
```

## 핵심 기능

### 1. 채팅 패널 (좌측)
- AI와 채팅하며 연구계획서 초안 요청
- 대화 히스토리 표시
- 입력창 + 전송

### 2. 초안 캔버스 (우측)
- 마크다운으로 렌더링된 연구계획서 초안 표시
- **영역 선택**: 텍스트 드래그로 영역 지정
- **컨텍스트 메뉴** (선택 시 표시):
  - **내용강화**: 채팅 입력이 열리고, 명령(예: "더 구체적으로 써줘")을 입력하면 해당 영역 강화
  - **삭제**: 선택 영역 삭제
  - **수정**: 수정 모달에서 직접 편집 후 저장
