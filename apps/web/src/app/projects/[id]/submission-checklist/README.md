# Submission Checklist (제출 체크리스트)

## 개요

연구계획서 **제출 전** 필수 항목을 점검하는 페이지입니다. RB 평가 검토 직후, 실행계획 생성 전에 위치하며, 연구자가 제출 준비가 완료되었는지 스스로 확인할 수 있습니다.

## 왜 이렇게 만들었는지

### 연구기획평가 흐름에서의 위치

- **RB 평가 검토** → 자가 점검으로 평가 기준 충족 여부 확인
- **제출 체크리스트** → 제출 서류·요건이 모두 갖춰졌는지 최종 점검
- **실행계획 생성** → 선정 시 실제 연구수행을 위한 일정 계획 (제출과는 별개)

제출 단계와 실행 단계를 분리하여, "지금 제출할 준비가 되었나"와 "선정되면 어떻게 진행할까"를 명확히 구분합니다.

### 체크리스트 항목

- 연구계획서 본문, 도식/표, 차별성 검토, 기술표준분류, Gantt 일정표 등 앞선 단계 결과물
- 예산서 첨부, 제출 서식 최종 점검

→ 공고별로 요구하는 서류가 다를 수 있으므로 추후 동적 로드 가능

### PDF 내보내기·제출하기

- PDF 내보내기: 연구계획서 통합본 생성 (추후)
- 제출하기: 통합공고 등 제출 API 연동 (추후)
