# Repository Agent Instructions

This repository maintains an OpenWiki-style documentation set under `openwiki/`.

Before making code changes, read:
- `openwiki/README.md` for the wiki map and current system summary.
- The relevant `openwiki/architecture/*` page for the area being changed.
- `openwiki/research/evidence-map.md` when changes affect the academic paper evidence trail.

After meaningful implementation changes, update:
- `openwiki/research/implementation-log.md` with what changed, why, files touched, and verification.
- `openwiki/research/evidence-map.md` if the change adds or changes paper evidence.
- The relevant architecture or decision page if system boundaries changed.

This wiki supports writing a paper about the process of building the AI Arch system. Do not treat it as an in-product paper generator.
